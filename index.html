Code Commenting Convention
v1.0

The Code Commenting Convention is a set of best practices designed to make code more understandable and maintainable. It emphasizes clear and concise comments that focus on explaining the reasoning behind the code rather than restating what the code does.


Usage:

<tag>[optional scope]: <description>
